ShelterSafe



ShelterSafe is an application that offers information on how to take safety measures in the event of a natural disaster. To do so, it displays a map, which displays the shelters close to the user, as well as information about that shelter. ShelterSafe will rely on radio communications as well as input from the user to transmit this information. Each shelter within a fifty mile radius would have a unique channel. This is to guarantee that each shelter can communicate data, such as location, available food, and capacity, to citizens without the risk of channel overlap.
